# Transit Data Notice

This directory contains processed public transportation data generated by
the transit station pipeline (`data/transit/`). See `sources.md` for
provenance details.

## GTFS Feeds

### Tokyo Metro

Tokyo Metro GTFS data is provided by Tokyo Metro Co., Ltd. via the
Public Transportation Open Data Center (ODPT) authenticated API.

When using, displaying, or redistributing data derived from this feed,
include attribution to:

- Public Transportation Open Data Center / Association for Open Data of Public Transportation
- Tokyo Metro Co., Ltd.

### Toei Subway

Toei Subway GTFS data is provided by the Tokyo Metropolitan Bureau of
Transportation via the ODPT public file API.

When using, displaying, or redistributing data derived from this feed,
include attribution to:

- Public Transportation Open Data Center / Association for Open Data of Public Transportation
- Tokyo Metropolitan Bureau of Transportation
